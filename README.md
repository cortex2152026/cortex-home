# Cortex Home

A lightweight static site:

- `/` → status + project intro
- `/learn` → the practical “how to use Cortex” guide (lead-dev mode)

## Local preview

```bash
cd cortex-home
python3 -m http.server 8080
```

## Deploy (Vercel)

```bash
npx vercel --prod
```

## Stack

- Static HTML/CSS
- Vercel hosting
